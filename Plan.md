## 목표 (이번 단계에서 구현할 동작)

**스페어(spare)** 프레임 다음에는 보너스로 다음 1굴림의 핀 수가 더해진다.

가장 단순한 스페어 케이스: 1프레임에서 5핀 + 5핀(스페어)을 치고, 다음 굴림(2프레임 첫 굴림)에서 3핀을 친 뒤, 나머지는 모두 거터(0핀)로 채운 게임의 점수는 **16점**이다.

- 1프레임: 5 + 5 (스페어) + 보너스 3 (다음 1굴림) = 13
- 2프레임: 3 + 0 = 3
- 나머지 프레임: 0
- 총합: 13 + 3 = 16

## 무엇을 테스트할 것인가

- `test_game.py`에 `test_spare_adds_next_roll_as_bonus` 테스트 추가

```python
def test_spare_adds_next_roll_as_bonus():
    game = Game()
    game.roll(5)
    game.roll(5)   # spare
    game.roll(3)
    game.roll(0)
    for _ in range(16):
        game.roll(0)
    assert game.score() == 16
```

## RED 검증 예상

- 현재 `score()`는 `sum(self.rolls)`이므로 이 테스트는 `5+5+3+0+...=13`을 반환해 `16`과 달라 실패한다.
- 즉 기대값(16)과 실제 구현 결과(13)의 차이(스페어 보너스 3점)가 분명히 드러나며, 진짜 RED다.
